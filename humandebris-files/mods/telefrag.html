Telefrag Railgun

Modification: Telefrag Railgun
Group: Human Debris
Modification Type: Server-Side
Quake III Arena Version: Q3A 1.27g
Source Code Included: Yes
Description: This is a mini-modification that changes the Quake3:Arena railgun into a telefrag gun.

In Depth:
One of my favorite features of Quake1 was the ability to telefrag other players when you spawned into the map. Quake3:Arena has much larger maps so telefragging does not occur as much as I'd like. This modification changes the railgun so that when it is fired at another player the player doing the firing telefrags the target. Telefragging other players means that players a always moving around, which makes the game play very fast and furious. Camping with a railgun is no longer an option.
The railgun has be modified so that the projectile stops after hitting one player. The reason for this is because telefragging 2 or 3 players in a row can cause a lot of blood and gibs (computer slow-down) and can be very disorienting to players.


Works With:
Deathmath: Yes
Team Detahmatch: Yes
Capture the Flag: Yes
Bots: Yes


Installation / Running:

  1. Unzip the archive into your quake3:arena directory.
  2. Load quake3 normally
  3. Select the "Telefrag Railgun" modification from the "Mods" menu.
  4. Load up a map and test it out.


Special Notes:
This is a server-side modification, which means if a server is running this game, clients can connect and play without having to download anything.
