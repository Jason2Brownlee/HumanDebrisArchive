I Love Rockets

Modification: I Love Rockets
Group: Human Debris
Modification Type: Server-Side
Quake III Arena Version: Q3A 1.27g
Source Code Included: Yes
Description: This modification changes Quake3 so that there are five rocket launchers to choose from.

In Depth:
All players spawn into the map with a rocket launcher and unlimited ammunition. There are no other weapons at all. There are five rocket launchers from which to choose. To select which rocket launcher you want to use for a game you must change the value of the 'g_rocket' cvar. Once this cvar has been changed the map must be restarted.
Note: Only the server can change the value of this cvar.

Here are a list of the five possible values for the cvar (five possible rocket launchers).
0 == Classic Rocket Launcher: This is the normal Quake3:Arena rocket launcher.
1 == Effected By Gravity: All rockets are effected by gravity.
2 == Homing Rockets: These are rockets that home in on their target.
3 == Rapid Fire Rockets: These are rockets that fire twice as fast as normal, but hurt half as much.
4 == Laser Guided Rockets: These are laser guided rockets that are very similar to the ones in Half-Life. The Crosshair is used as the laser. Which ever direction you point it, the rocket will head towards.


Works With:
Deathmath: Yes
Team Detahmatch: Yes
Capture the Flag: Yes
Bots: Yes


Installation / Running:

  1. Unzip the archive into your quake3:arena directory.
  2. Load quake3 normally
  3. Select the "I Love Rockets" modification from the "Mods" menu.
  4. Load up a map and test it out.


Special Notes:
This is a server-side modification, which means if a server is running this game, clients can connect and play without having to download anything.
