Quake3 2D Platform Game

Modification: Quake3 2D Platform Game
Group: Human Debris
Modification Type: Client/Server
Quake III Arena Version: Q3A 1.27g
Source Code Included: Yes
Description: This modification transforms Quake3 into a 2D style platform game.

In Depth:
Movement is restricted to forward, backwards, duck and Jump, lookup and down. Here is the recommended control setup:
D Move Right
A Move Left
Mouse1 Fire Weapon
Alt Duck
Space Bar / Mouse2 Jump
Shift (Right Shift) Taunt
Q Change Weapon
Mouse Look up and down as well as turn around

Of course you can set the keys to anything you want.

To load the q3pg test map, select them from the skirmish menu (submenu of singleplayer). Additional maps that are downloaded can be accessed in the skirmish menu.


Making Maps:
Making maps for this mod is really simple. When you download this mod, an example .MAP file is included in the archive. Simply load it up in q3Radient and see how it was done. You can use the example map as a base if you want. If you have made a map, package it into a .PK3 with a levelshot, .aas file, and .arena file and post a message on our forum or email us so we can add it to the downloads section.

If you are unsure how to package it up email the map to us and we will package it up for you.


Works With:
Deathmath: Yes
Team Detahmatch: Yes
Capture the Flag: Yes
Bots: Yes


Installation / Running:

  1. Unzip the archive into your quake3:arena directory.
  2. Load quake3 normally
  3. Select the "2D Platform Game" modification from the "Mods" menu.
  4. Load up the test map and test it out.


Special Notes:
This is a client/server modification, which means if a server is running this modification the client needs to download the modification as well.
