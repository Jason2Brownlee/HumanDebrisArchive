Sog's Rifle

Modification: Sog's Rifle
Group: Human Debris
Modification Type: Client/Server
Quake III Arena Version: Q3A 1.27g
Source Code Included: Yes
Description: This modification changes the Quake3:Arena so that all players only have a high powered rifle and low amounts of ammunition.

In Depth:
All players spawn into the map with a rifle and ten rounds of ammunition. There is no health, armor, powerups or ammunition. All weapons have been replaced with rifles, and each additional rifle that is picked up contains two rounds of ammunition.

This is a game of skill where every shot counts. Included in this modification is a new rifle model and new firing sounds.


Works With:
Deathmath: Yes
Team Detahmatch: Yes
Capture the Flag: Yes
Bots: Yes


Installation / Running:

  1. Unzip the archive into your quake3:arena directory.
  2. Load quake3 normally
  3. Select the "Sog's Rifle" modification from the "Mods" menu.
  4. Load up a map and test it out.


Special Notes:
This is a client/server modification, which means if a server is running this modification the client needs to download the modification as well.
